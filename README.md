Projekt SCADA – Etap II (Python):
Aplikacja wizualizująca proces napełniania zbiorników.
Projekt wykorzystuje PyQt5 oraz architekturę obiektową.

Funkcje:
- Wizualizacja poziomu cieczy
- Sterowanie zaworami
- Automatyczny proces
- Wiele zbiorników

Uruchomienie -- python main.py
